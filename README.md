# FileTransferTCP-UDP
To compile:
	- Open terminal in the working directory
	- Run the command "make"

To clean:
	- Open terminal in the working directory
	- Run the command "make clean"

To start sender:
	- Compile the program
	- Open terminal in the working directory
	- Run the command "java SenderGui"

To start receiver:
	- Compile the program
	- Open terminal in the working directory
	- Run the command "java ReceiverGUI"
